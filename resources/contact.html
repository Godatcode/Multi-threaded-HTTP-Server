API Endpoints

This server provides the following HTTP endpoints for testing and demonstration purposes.

GET Endpoints

Retrieve HTML pages and download binary files

GET

/

Returns the index.html page (home page)

curl http://localhost:8080/
GET

/about.html

Returns the about page

curl http://localhost:8080/about.html
GET

/logo.png

Downloads the logo.png file as binary

curl -O http://localhost:8080/logo.png
GET

/sample1.txt

Downloads a text file as binary

curl -O http://localhost:8080/sample1.txt

POST Endpoints

Upload JSON data to the server

POST

/upload

Accepts JSON data and creates a file in resources/uploads/

Request Headers:

Content-Type: application/json Host: localhost:8080

Request Body:

{ "name": "Test User", "email": "test@example.com", "message": "This is a test message" }

Example with curl:

curl -X POST http://localhost:8080/upload \ -H "Content-Type: application/json" \ -H "Host: localhost:8080" \ -d '{ "name": "Test User", "email": "test@example.com", "message": "Hello from HTTP Server!" }'

Response (201 Created):

{ "status": "success", "message": "File created successfully", "filepath": "/uploads/upload_20240315_123456_a7b9.json" }

Error Responses

400 Bad Request

Malformed request or missing Host header

403 Forbidden

Unauthorized path access or Host mismatch

# Example: Path traversal attempt curl http://localhost:8080/../../../etc/passwd # Response: 403 Forbidden

404 Not Found

Requested resource doesn't exist

curl http://localhost:8080/nonexistent.html # Response: 404 Not Found

405 Method Not Allowed

HTTP method not supported (PUT, DELETE, etc.)

curl -X PUT http://localhost:8080/index.html # Response: 405 Method Not Allowed # Allow: GET, POST

415 Unsupported Media Type

Wrong Content-Type or unsupported file type

# Example: POST with wrong content type curl -X POST http://localhost:8080/upload \ -H "Content-Type: text/plain" \ -d "This is plain text" # Response: 415 Unsupported Media Type

Python Test Script

Here's a Python script to test all endpoints:

import requests import json # Test GET request response = requests.get('http://localhost:8080/') print(f"GET /: {response.status_code}") # Test binary file download response = requests.get('http://localhost:8080/logo.png') print(f"GET /logo.png: {response.status_code}, Size: {len(response.content)} bytes") # Test POST request data = { "name": "Test User", "email": "test@example.com", "message": "Hello from Python!" } response = requests.post( 'http://localhost:8080/upload', headers={'Content-Type': 'application/json'}, data=json.dumps(data) ) print(f"POST /upload: {response.status_code}") print(f"Response: {response.json()}") # Test security - path traversal response = requests.get('http://localhost:8080/../etc/passwd') print(f"GET /../etc/passwd: {response.status_code} (should be 403)") # Test unsupported method response = requests.put('http://localhost:8080/index.html') print(f"PUT /index.html: {response.status_code} (should be 405)")
